Repo A = Repository yang dijadikan patokan
Repo B = Repository kita (hasil forking dari repo A)

1. Fork repo A (di github.com)

2. Clone repo B ke lokal (Git clone url_repo_B)

3. Remote repo A ke lokal (Git remote add nama_remote url_repo_A)
 repo ghazy di dalem folder clone
4. Fetch repo A (Git fetch nama_remote_ke_repo_A)
5. Lanjutannkalau dah diupdate
- Git fetch nama_remote_ke_repo_A
- Merge repo A (Git merge nama_remote/branchnya) branchnya = update
- Push ke remote repo B (Git push -u origin master)
